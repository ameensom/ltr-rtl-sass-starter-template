SASS RTL/LTR Template

What does it do?

This template works a skeleton for creating web interface with dual direction, you will need to compile once and use the same scss files to style the page.
Just follow the simple guidelines and your work will be done in not time.

Features

- Ability to add more than one language.
- browserSync integration that enables auto refresh of the browser upon saving html/scss/js files.
- Cool mixins included.
- Easy to read and understand scss file structure.
- Added autoprefixer support.

How to use ?

- Start by defining the language that you are going to use.

Click on the interface local you want to preview/debug/test :
